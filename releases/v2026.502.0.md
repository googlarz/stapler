# v2026.502.0

> Released: 2026-05-02

## Highlights

- **npm publishing** — `@googlarz/stapler` CLI package is now published to npm on every stable release. Install with `npm install -g @googlarz/stapler`.
- **Canary releases** — Canary builds are automatically published to npm on every push to main with the `canary` dist-tag.

## Improvements

- **CI release pipeline** — Release workflow now correctly publishes only the public `@googlarz/stapler` CLI package; all internal `@stapler/*` packages are marked private to prevent unauthorized publish attempts.
- **Test reliability** — Path resolver tests rewritten to use dependency injection instead of `vi.mock("node:fs")`, eliminating flaky teardown behavior in CI.
- **Docker build** — Fixed `@stapler/adapter-utils` missing from Docker build dependencies.

## Fixes

- **Skill invocation fire-and-forget** — Wakeup failures in skill invocations no longer propagate the error to callers; the `skill_invocations` row captures the failure status.
- **Server startup env var** — `STAPLER_API_URL` environment variable is now correctly used (renamed from `STAPLER_API_URL`).
